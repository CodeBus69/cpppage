Środowisko programistyczne

Kod C++ możemy napisać nawet w notatniku, jednak aby go skompilować potrzebujemy tzw. kompilatora. W naszej szkole używamy Code::Blocks i na takim też polecam ćwiczyć na początku. Nie wymaga on praktycznie żadnej konfiguracji, aby zacząć na nim działać. Jeśli chcesz kompilować kod bez pobierania żadnego IDE (środowiska programistycznego), możesz skorzystać z takich stron jak cpp.sh

Poradnik, jak zainstalować Code:Blocks możesz znaleźć tutaj

Struktura programu

Instalując Code::Blocks i tworząc nowy projekt otrzymujemy przykładowy kod, który wygląda w następujący sposób

#include <iostream>
using namespace std;
int main ()
{
cout << "Hello world!" << endl;
return 0;
}

Omówmy teraz każdą linijkę tego kodu

#include<iostream>

W tej linijce deklarujemy bibliotekę, jaką będziemy chcieli używać w naszym programie. Chodzi o to, że dana biblioteka posiada swoje funkcje i zmienne które nie zawsze musimy potrzebować. Między ostrymi nawiasami jest nazwa biblioteki w tym wypadku iostream
Oznacza ona input/output stream czyli strumień wejścia i wyjścia, dzięki niej możemy uzywać strumienia cout i cin, które omówimy w dalszej części

using namespace std;

Nie rozwodząc się zbyt długo, dzięki tej linijce nie będziemy musieli pisać za każdym razem std::cout czy std::cin, tylko skróci nam to zapis do cout i cin

int main ()

Jest to funkcja main, w późniejszej części kursu dowiemy się, że program to zbiór rożnych funkcji, które ze sobą współdziałają. Funkcja main() jest funkcją główną w języku C++ i program zawsze zaczyna działanie od niej, nie może się ona nazywać inaczej. Funkcja zawiera klamerki { } wewnątrz których znajduje się kod wykonywany przez daną funkcję

cout << "Hello world!" << endl;

Jest to strumień wyjścia, czyli strumień kierujący dane na zewnątrz programu, w tym wypadku na ekran konsoli. Operatory << skierowane w lewo mówią nam o tym, że strumień danych jest wychodzący. Dzięki tej linijce kodu na ekranie wyświetli nam się komunikat Hello world! znajdujący się między nawiasami, element endl powoduje przejście do następnej linijki

return 0;

W dziale z funkcjami dowiesz się, że każda funkcja może zwrócić jakąś wartość, w tym wypadku za pomocą tej linijki funkcja main() zwraca wartość 0 nie musisz się na razie tym przejmować, dokładne rozumienie funkcji dopiero w przyszłych rozdziałach

Zmienne

Istotnym elementem praktycznie każdego języka programowania są zmienne, czyli 'komórki' w których program przechowuje dane. W C++ mamy kilka rodzajów zmiennych. Tymi podstawowymi są:

  • int - liczby całkowite
  • string - wyrazy i zdania
  • char - pojedyńcze znaki
  • double - liczby zmiennoprzecinkowe

Zmienna potrzebuje zadeklarowania, aby program zarezerwował dla niej miejsce w pamięci, nie możemy więc wykorzystywać zmienych, które nie istnieją.

int wiek;
string imie;

Możemy również od razu przypisać im wartości

int wiek = 18;
string imie = "Marcin";

Wykorzystywanie zmiennych

Najprostszym sposobem na wykorzystanie zmiennej jest jej wyświetlenie na ekranie, możemy to zrobić komendą poznaną wcześniej

int wiek = 18;
string imie = "Marcin";
cout << imie << " ma " << wiek << " lat " << endl;

Ta komenda spowoduje wyświetlenie się na ekranie napisu
Marcin ma 18 lat
Zauważ, że jeśli chcemy użyć wartości danej zmiennej, to nazwę zmiennej wpisujemy bez cudzysłowia, ponieważ jeśli byśmy tak zrobili, czyli:

int wiek = 18;
string imie = "Marcin";
cout << "imie" << " ma " << "wiek" << " lat " << endl;

To ujrzelibyśmy na ekranie następujące zdanie:
imie ma wiek lat

Dzieje się tak ponieważ, gdy wpisujemy pewną wartość w cudzysłów, to program bierze pod uwagę to co jest dosłownie między nawiasami. Wpisanie bez nawiasów sprawia, że program zaczyna poszukiwać zmiennej o takiej nazwie i ją używa.
